ATIF SALMAN SHEIKH — PERSONAL WEBSITE
=====================================

This is a self-contained static website. No build step, no dependencies.

FILES
  index.html                     → homepage (open this to preview locally)
  style.css                      → all styling
  articles/supply-stress.html    → full article
  articles/food-delay.html       → full article
  assets/                        → photos + CV PDF

PREVIEW LOCALLY
  Just double-click index.html — it opens in any browser.

PUBLISH (free options, pick one)
  • GitHub Pages: create a repo, upload these files, enable Pages on the main branch.
  • Netlify: drag this whole folder onto app.netlify.com/drop — instant live URL.
  • Vercel: `vercel` in this folder, or import the repo at vercel.com.
  Any of these gives you a shareable link (and you can add a custom domain later).

EDITING
  • Update text directly in the .html files.
  • Replace assets/atif.jpg or assets/ksbl.jpg to change photos.
  • Replace assets/Atif_Salman_Sheikh_CV.pdf to update the CV download.

NOTES
  • The Medium article ("The Context Trap") links out to Medium by design.
  • The two marketplace articles are hosted here in full.
  • Social-share preview (OG tags) uses your photo; update the og:image URLs
    to an absolute https:// URL once you know your final domain.
